This sample program demonstrates usage for the following features:

·         Producing a trace file with debug output.

·         Calling a stored procedure that uses input and output parameters and has a return value.

·         Passing parameters using the two supported methods (named parameters and parameter markers).

 

Please read Chapter 2 "Using the Adaptive Server Enterprise .NET Data Provider Sample Applications" for detailed explanation of the code and instructions on how to use this sample application.

 

In order to run this sample program, you will need either Microsoft Visual Studio .NET 2003 or .NET Framework v1.1 installed on the system. You will also need access to an Adaptive Server with “pubs2” sample database installed.

 

Instructions for Visual Studio .NET:

 

1)    Open the “Advacned” project in Visual Studio.

2)    If the ASE .NET Data Provider has not been properly installed on this system, please refer to the User Guide for instructions on how to fix the reference to ASE .NET Data Provider in the sample project.

3)    Press Ctrl+F5 key to build and run the sample program.

 

If you do not use Microsoft Visual Studio .NET you can compile this sample using .NET Framework version 1.1 as follows:

 

1)    Add the directory with .NET Framework v1.1 binaries to your system path.

2)    Check to make sure that the “dll” directory under the ASE .NET Provider installation is included in the system path and the “LIB” environment variable (the default value for this directory is C:\Sybase\ADO.NET\DLL).

3)    Compile the sample program using the supplied build script “build.bat”.

4)    To run the program, type “advanced.exe”
