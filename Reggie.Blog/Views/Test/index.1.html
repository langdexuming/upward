什么是MVVM设计模式？

XAML 通过将应用程序的GUI与编程逻辑（在C #or VB.NET中编码）分离，在WPF，Silverlight，Windows Phone和Windows 8应用程序中启用 SoC。Model-View-ViewModel（MVVM）是一种设计模式，通过允许将Business逻辑与View（UI）分开来解决 SoC，最终使得更容易编写单元测试并实现并行开发和设计。它利用XAML平台的丰富数据绑定功能，通过视图的DataContext属性将视图模型公开到UI 。业务层也被称为模型，而ViewModel层负责使用DataBinding将模型中的数据对象暴露给UI。该ViewModel包含View显示逻辑，可以使用ViewModel中声明的命令属性来处理UI上的操作。

什么是.NET Core？

.NET Core 是一个通用开发平台，由 Microsoft 和 GitHub 上的 .NET 社区共同维护。 它是跨平台的，支持 Windows、macOS 和 Linux，并且可用于设备、云和嵌入式/IoT 方案。

JavaScript介绍

JavaScript一种直译式脚本语言，是一种动态类型、弱类型、基于原型的语言，内置支持类型。它的解释器被称为JavaScript引擎，为浏览器的一部分，广泛用于客户端的脚本语言，最早是在HTML（标准通用标记语言下的一个应用）网页上使用，用来给HTML网页增加动态功能。

ASP.NET Core介绍

ASP.NET Core是一个跨平台，高性能，开源的框架，用于构建基于云的互联网应用程序。
